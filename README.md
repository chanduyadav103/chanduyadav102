 **How to Run Locally**

**Step: 1. Clone the repo**

```bash
git clone https://github.com/chanduyadav103/Chandu-s-Portfolio.git
```

**Step: 2. Open the folder**

```bash
cd Chandu-s-Portfolio
```

 **Step: 3. Run the project**

You can simply open **index.html** in your browser.

Or use Live Server:

```bash
npx live-server
